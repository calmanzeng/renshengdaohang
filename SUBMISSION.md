# Spark Hackathon 提交材料

## 项目名称
Ziwei Destiny SBT — 紫微命盘区块链存证

## 一句话描述
将中国传统紫微斗数命盘数据铸造为 Soulbound NFT，永久不可篡改地存证于 Monad 区块链。

## Problem
传统命盘数据存储在中心化服务器，存在被篡改、丢失的风险。

## Solution
用户排盘后，将出生年/月/日/时辰/命盘指纹哈希上链铸造为 Soulbound NFT，不可转移、不可篡改、每人仅限一次。

## 技术创新
- Monad Mainnet (143) 存证
- Soulbound ERC-721 变体: 禁止所有转移操作
- On-chain Base64 metadata
- 合约仅 8.9KB，Gas 优化
